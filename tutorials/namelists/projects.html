Project folders

As the namelists are often updated between each release of COMPS, we recommend placing custom declarations in a separate folder. The standard declarations that come shipped with the software are located in ./namelists/default/.

.

As an example, we will create the folder ./namelists/custom/. This folder can contain the same files as you find in ./namelists/default/. To refer to lines in these files, the folder name followed by a period is added in front of the names. Forexample, if a downscaler called "nearest" is defined in ./namelists/custom/schemes.nl, it can be refered in any configurations.nl as custom.nearest. The same applies to configurations, variable-configurations, and runs defined in this folder.
