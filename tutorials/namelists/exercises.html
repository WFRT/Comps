Excercises

Alter the tutorial run:

1) Set up your own folder of files

Create a new folder called 'custom' in the namelists folder. Duplicate the runs.nl and varconfs.nl files from the default folder (you do not need to duplicate any other namelist files). From here on you will use the tutorial run defined in this folder (i.e. referenced by custom.tutorial).

2) Climatology configuration

Add the climatology configuration to the tutorial run, by editing namelists/custom/varconfs.nl. Rerun COMPS to produce the climatology forecasts. How does the climatology forecasts compare to the persistence forecasts?

3) Quantile-quantile

Create a new configuration in namelists/custom/configurations.nl using the quantile-quantile correction scheme. Use reasonable schemes for the other components. The quantile-quantile correction scheme is already set up in the default schemes namelist. Add this configuration to your tutorial runi by editing namelists/custom/varconfs.nl. Rerun COMPS using this configuration. How does these forecasts compare to the Kalman filter forecasts?

4) Altering the Kalman filter ratio

Copy the kf scheme from the default namelist into the schemes namelist in your own folder. Alter the ratio option for the scheme to 1. Create a new configuration in your own folder that uses this new Kalman filter scheme. Rerun COMPS using this configuration and comment on the results.
