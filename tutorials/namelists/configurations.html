Configurations

Configurations are defined in ./namelists/default/configurations.nl. A configuration is declared by specifying the following components:
ComponentPurpose
selector Constructs ensemble from input data
correctors
continuous
lowerDiscrete
upperDiscrete
calibrators
averager
Required: will not run without these
Optional.

As an example, consider the following configuration declaration: {% highlight python %} clim selector=clim downscaler=nearest averager=mean continuous=bpe {% endhighlight %} Here the clim selection scheme, the nearest downscaler, the mean averager, and the bpe continuous probability model has referenced from ./namelists/default/schemes.nl.
