View results

COMPS generates output files for each day, which is stored in results/tutorial/. The tutorial run is set-up to produce NetCDF files, which are found in results/tutorial/netcdf/. COMPS can be set-up to produce files in many different formats. Therefore, Files for each output type is located in its own folder. For the tutorial run, the only output type is netcdf/.

Plotting with meteo

COMPS has a tool called meteo that draws meteograms of forecasts. It is found in {% highlight bash %} graphics/meteo results/tutorial/netcdf/20120101_0_T_kf.nc 0 {% endhighlight %} The last argument is the station number to produce forecasts for, in this case 0. The tool shows:
  • The forecast distribution over time
  • The observations

Inspecting with ncdump

{% highlight bash %} ncdump results/tutorial/netcdf/20120101_0_T_kf.nc | less {% endhighlight %}
