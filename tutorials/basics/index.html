Running COMPS

This tutorial assumes COMPS has been installed.

This tutorial will produce forecast and compare it against a persistence forecast. The COMPS program is executed as follows: {% highlight bash %} $ ./comps.exe 20120101 20120301 tutorial {% endhighlight %} where the first parameters (20120105) is a date in YYYYMMDD format and the second (tutorial) is a string specifying which run to execute.

This will complete the tutorial run, which creates temperature forecasts for three configurations: raw, persistence, and kalmanFilter.
