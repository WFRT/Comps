Running COMPS

This tutorial assumes COMPS has been installed.

This tutorial will produce forecast and compare it against a persistence forecast. The COMPS program is executed as follows: {% highlight bash %} $ ./comps.exe 20120101 20120301 tutorial {% endhighlight %} where the first parameters (20120101) is start date (in YYYYMMDD format), the second parameter (20120301) is the end date, and the third (tutorial) is a string specifying which run to execute.

This will complete the tutorial run, which creates temperature forecasts for three configurations: raw, persistence, and kalmanFilter.
