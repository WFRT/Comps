Get started

This page will help you download and install COMPS.

1) Pre-requisites

COMPS uses a few commonly used tools/libraries:
LibraryPurpose
SCons Tool used to compile COMPS
Boost (v1.49) Boost provides many useful statistical functions
NetCDF (v3.6) Read/write NetCDF files. Note both libnetcdf and libnetcdf_c++ are needed.
grib_api Read GRIB input files
ncursesAlternate user interface

Required: will not compile without these
Optional: will run, but some features disabled

Setup on Ubuntu

{% highlight bash %} $ sudo apt-get install scons $ sudo apt-get install libnetcdf-dev $ sudo apt-get install netcdf-bin $ sudo apt-get install libboost_filesystem-dev $ sudo apt-get install libboost_system-dev $ sudo apt-get install grib_api $ sudo apt-get install python-numpy $ sudo apt-get install python-scipy $ sudo apt-get install python-matplotlib {% endhighlight %}

2) Download

Download the latest code in zip format and unzip the contents:

Download

Alternatively, get the latest version through git:

{% highlight bash %} $ git clone git://github.com/wfrt/Comps.git comps {% endhighlight %}

3) Installation

In the Comps directory, execute the following: {% highlight bash %} $ scons {% endhighlight %} The installation should produce the executable comps.exe
