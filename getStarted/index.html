Get started

Pre-requisites

COMPS has been tested on:
Operating systemsVersions
Ubuntu8.04
CentOS6.4
Mac10.7 (Lion)
COMPS uses a few common libraries:
LibraryPurpose
*Boost (v1.49) Boost provides many useful statistical functions
*NetCDF (v3.6) Read/Write NetCDF. Note both libnetcdf and libnetcdf_c++ are needed.
grib_api/OpenJPEG Read GRIB input files
NcursesAlternate user interface
*required

Download

Download the latest code in zip format

Download

Alternatively, get the latest version through git:

{% highlight bash %} $ git clone git://github.com/wfrt/Comps.git comps {% endhighlight %}

Installation

{% highlight bash %} $ cd comps $ ./configure $ make {% endhighlight %} The installation should produce the exeutable comps.exe
