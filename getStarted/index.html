Download

Alternatively, get the latest version through git:

{% highlight bash %} $ git clone git://github.com/tnipen/Comps.git comps {% endhighlight %}

Install

{% highlight bash %} $ cd comps $ ./configure $ make {% endhighlight %} The installation should produce the exeutable comps.exe

Running

Set up data

Every input dataset has its own directory within input/. To add your own data follow the instructions in input/test/README. GFS data can be downloaded as follows: {% highlight bash %} $ cd input/gfs $ ./download 20130101 {% endhighlight %} A global observation dataset can be downloaded using a similar procedure in input/rda336/
Once the input data has been set up, it is time to run the experiment: {% highlight bash %} $ ./driverO.exe 20130101 test {% endhighlight %}
